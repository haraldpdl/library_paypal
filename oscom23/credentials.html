Manage API Credentials PayPal: Credentials

The PayPal account API Credentials can be manually entered on this page or can be automatically retrieved using the Retrieve API Credentials buttons.

The API Credentials consists of:

  • API Username
  • API Password
  • API Signature
  • Merchant Account ID
  • PayPal E-Mail Address

The API Credentials are used to securely communicate with PayPal and perform transactions with.

If PayPal payments are to be processed through Payflow, the Payflow credentials can be entered using the Payflow button.
